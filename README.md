DHCP DNS WEB SERVER NETWORK

1 Router (DHCP)

## DNS Server Configuration
On the server, choose Server → DNS.
Enable DNS service.
Add host entries:
Hostname	         |     IP Address
www.isaacuko.com	 |    192.168.20.2
